# database_port

## Synopsis

`database_port` = _port_

_Port_ is the number of the port the Supervisor will use to connect to the database server.

## Example
```
database_port = 5001
```

## Defaults

### PostgreSQL (Qube 7 onwards)

50055

### MySQL (Qube 6 and older)

**Linux / macOS** : 3306

**Windows** : 3300
