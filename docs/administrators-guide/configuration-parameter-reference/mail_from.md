# mail_from

## Synopsis

`mail_from` = _user_

_User_ is the mail user name or address that the Supervisor will use in the "From" field of notification email that it sends to users or the admin.

## Example

```
mail_from = qubeadmin  
mail_from = qubeadmin@mydomain.com
```

## Defaults

qube
