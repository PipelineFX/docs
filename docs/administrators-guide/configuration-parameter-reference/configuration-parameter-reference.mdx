# Configuration Parameter Reference

This page provides documentation for all of Qube!'s configuration parameters.

## Local Configuration
The Configuration for the Supervisor, Worker, and Clients can be found in the
qb.conf file that is located at the following locations depending on the
platform. It can be configured directly with a text editor or through the
QubeUI's [Administration Panel](/Documentation/users-guide/qube-ui/panels/Administration+Panel). 

| OS                  | Path                            |
| --------------------|---------------------------------|
| **Linux & macOS:**  | /etc/qb.conf                    |
| **Windows 10/11:**  | C:\ProgramData\pfx\qube\qb.conf |

## Central Worker Configuration
On the Supervisor, there is an additional optional configuration file
qbwrk.conf for centrally modifying the Worker configurations. It can be
modified directly with a text editor or through the WranglerView's Host/Worker
Layout right-click menu item "Configure On Supervisor..." for the selected Workers.
