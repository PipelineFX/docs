# Advanced User's Guide

## Qube! Advanced User's Guide
This guide is aimed at advanced users and render wranglers - people who manage other people's jobs and who monitor the farm to make sure it is performing well. 

* [Job Reference](job-reference) - Explains the details of what a job is in Qube!

* [Command Line Reference](command-line-reference) - Users can also submit and manage jobs via the command line.

## See Also
For users submitting and managing their own jobs, see the [User's Guide](users-guide) documentation..

The [Administrator's Guide](administrators-guide) discusses installing and tuning Qube! for specific deployments.

TDs and other technical users can create their own jobtypes and also script customizations of Qube!. These actions are explained in the [Developer's Guide](developers-guide).
