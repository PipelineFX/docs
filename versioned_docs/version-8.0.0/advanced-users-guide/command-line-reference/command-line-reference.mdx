# Command Line Reference

## All commands
| Command | Description |
| ---     | ---         |
| [qbadmin](advanced-users-guide/command-line-reference/qbadmin) | Administrative command used to control qube |
| [qbarchive](advanced-users-guide/command-line-reference/qbarchive) | Output a job to a Qube archive file. |
| [qbblock](advanced-users-guide/command-line-reference/qbblock) | Forces a job to quit, and suspend its execution on the farm |
| [qbbottom](advanced-users-guide/command-line-reference/qbbottom) | Move a job to the bottom of its priority level |
| [qbconfigfile](advanced-users-guide/command-line-reference/qbconfigfile) | Parse a config file and display results |
| [qberr](advanced-users-guide/command-line-reference/qberr) | Display the stderr log of a job instance |
| [qbhash](advanced-users-guide/command-line-reference/qbhash) | Generate and display the hashed string for a given password |
| [qbhist](advanced-users-guide/command-line-reference/qbhist) | Display the history of a job |
| [qbhosts](advanced-users-guide/command-line-reference/qbhosts) | List Worker hosts |
| [qbinterrupt](advanced-users-guide/command-line-reference/qbinterrupt) | Force a job to quit, and requeue immediately in a pending state |
| [qbjoborder](advanced-users-guide/command-line-reference/qbjoborder) | List jobs scheduled to run on a Worker host |
| [qbjobs](advanced-users-guide/command-line-reference/qbjobs) | List jobs |
| [qbkill](advanced-users-guide/command-line-reference/qbkill) | Kill jobs |
| [qblock](advanced-users-guide/command-line-reference/qblock) | Lock available job slots on a Worker |
| [qblogin](advanced-users-guide/command-line-reference/qblogin) | Windows passwords management |
| [qbmigrate](advanced-users-guide/command-line-reference/qbmigrate) | Force a job to release its host and move to another |
| [qbmodify](advanced-users-guide/command-line-reference/qbmodify) | Modify jobs in the queue |
| [qbout](advanced-users-guide/command-line-reference/qbout) | Display the stdout log of a job instance |
| [qbpreempt](advanced-users-guide/command-line-reference/qbpreempt) | Ask a job to release its worker job slots, let the current work (frame) finish |
| [qbremove](advanced-users-guide/command-line-reference/qbremove) | Remove jobs from the queue |
| [qbrequeue](advanced-users-guide/command-line-reference/qbrequeue) | Moves a job into the blocked state after a job has completed |
| [qbresume](advanced-users-guide/command-line-reference/qbresume) | Resume a suspended job |
| [qbretire](advanced-users-guide/command-line-reference/qbretire) | Ask a job instance to release its host, and then go prematurely to "complete" state |
| [qbretry](advanced-users-guide/command-line-reference/qbretry) | Moves a job from a "done" (complete/failed/killed) status back to pending |
| [qbshove](advanced-users-guide/command-line-reference/qbshove) | Force the Supervisor to re-evaluate a job immediately |
| [qbstats](advanced-users-guide/command-line-reference/qbstats) | Display job statistics |
| [qbsub](advanced-users-guide/command-line-reference/qbsub) | Submit a cmdline or cmdrange job |
| [qbsummary](advanced-users-guide/command-line-reference/qbsummary+) | Display queue summary |
| [qbsuspend](advanced-users-guide/command-line-reference/qbsuspend) | Suspend a process upon the host which the job is running on |
| [qbtop](advanced-users-guide/command-line-reference/qbtop) | Move a job to the top of its priority level |
| [qbunblock](advanced-users-guide/command-line-reference/qbunblock) | Move a job from a blocked state into the pending state |
| [qbunlock](advanced-users-guide/command-line-reference/qbunlock) | Unlock available job slots on a Qube! Worker |
| [qbupdateresource](advanced-users-guide/command-line-reference/qbupdateresource) | Inform the supervisor of license usage counts from external license managers |
| [qbusers](advanced-users-guide/command-line-reference/qbusers) | Modifies the permissions for an individual user |
