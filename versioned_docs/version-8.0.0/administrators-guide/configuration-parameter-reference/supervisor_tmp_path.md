# supervisor_tmp_path

## Synopsis

`supervisor_tmp_path` =  _path_

_Path_ is used to specify where the Supervisor will write its temporary files.

## Example

```
supervisor_tmp_path = /tmp
```

## Defaults

OS | Path
---|---
Linux & macOS: | /tmp
Windows: | (Windows Default User Temp Path)
