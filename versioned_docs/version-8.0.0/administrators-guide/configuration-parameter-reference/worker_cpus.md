# worker_cpus

## Synopsis

`worker_cpus` = _CPUs_

_CPUs_ is the number of CPUs the Worker will report to the Supervisor. A
value of 0 tells Qube! to set the number of CPUs equal to the number of cores
seen by the OS.

## Example

```
worker_cpus = 2
```

## Defaults

0
