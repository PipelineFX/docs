# Common Supervisor Actions

The qbadmin command is a useful tool for a number actions for the Supervisor. It can provide information about the Supervisor, suspend job dispatch service, and command the Supervisor to reread and pass on configuration information to the Workers. The qbadmin command is located in the sbin directory of the Qube! distribution.

- Linux:    /usr/local/pfx/qube/sbin/qbadmin
- macOS:    /Applications/pfx/qube/sbin/qbadmin
- Windows:  C:\Program Files\pfx\qube\sbin\qbadmin.exe
