# Administrator's Guide

## Qube! Administrator's Guide
This guide is aimed at Administrators - people who install and maintain the Qube! software, and who configure it to maximize performance for their facilities.

[Additional Install Information](additional-install-information) - Installs are covered in the Start Here section, but this gives additional detail, including manual install procedures

[Managing Qube!](managing-qube) - Common operations for the Supervisor and Workers

[Configuring Qube!](configuring-qube) - Details of what can be configured, and how, for optimal performance

[Config Parameter Reference](configuration-parameter-reference) - All of the available configuration parameters and their potential values


For users submitting and managing their own jobs, see the [User's Guide](users-guide) documentation.

The [Advanced User's Guide](advanced-users-guide) discusses managing jobs and other advanced user topics.

TDs and other technical users can create their own jobtypes and also script customizations of Qube!. These actions are explained in the [Developer's Guide](developers-guide).
